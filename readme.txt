VendesFácil es un servicio de Coodinadora, **Logística de Transporte**

== Description ==

VendesFácil es un servicio de Coordinadora, este plugin integra su servicio **Logística de Transporte**
que permite a un comercio fácilmente gestionar todo el proceso de transporte de un producto de manera
automatizada, permitiendo la cotización del flete, solicitud de recogida, transporte,
seguimiento en línea (Tracking) y solución de novedades.

== Installation ==

1. Descargar el plugin
2. Ingrese a su administrador de wordpress
3. Ingrese al menu plugins / añadir nuevo
4. Busque por el nombre **Shipping VendesFacil Woocommerce**
5. Instale y active el plugin y siga el proceso de configuración.
6. Establezca los detalles del envío del producto, ver captura de pantalla.
